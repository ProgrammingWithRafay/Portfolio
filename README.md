# Rafay - Personal Portfolio

A modern, interactive portfolio showcasing my work as a Full Stack Developer and Machine Learning Engineer. The design for this portfolio was created using Figma, and it is built to be highly responsive, performant, and visually striking.

## Features

- Interactive 3D SVG Character: A custom-built, purely mathematical SVG illustration that actively tracks the user's cursor across the screen.
- Cinematic Loading Screen: A custom pre-loader featuring a pitch-black background with pulsing typography and an animated curtain reveal.
- Scroll-Reveal Animations: Lightweight, native IntersectionObserver-driven fade-up animations and staggered cascades.
- Premium Dark Aesthetic: Space-themed interface built using pure CSS, Tailwind, and custom cubic-bezier transitions.
- Highly Performant: Built natively in React and Vite without the bloat of heavy third-party animation libraries.

## Technology Stack

- React
- Vite
- TypeScript
- Tailwind CSS

## Running Locally

To run this project locally on your machine:

1. Clone the repository:
   ```bash
   git clone https://github.com/ProgrammingWithRafay/Portfolio.git
   cd Portfolio
   ```

2. Install the dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```
